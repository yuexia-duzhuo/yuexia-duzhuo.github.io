Hexo从0到1搭建个人博客

发表于2024-08-06|更新于2024-08-08
|字数总计:1.1k|阅读时长:3分钟|阅读量:

一，搭建前的软件准备（git，node）

搭建之前需要准备的软件：
Git：官网下载： https://git-scm.com/
Node.js 官网下载： http://nodejs.cn/

补：git是一个工具，作用是管理不同版本的代码

二， 安装hexo，完成简单本地页面展示

  1. 进入cmd窗口输入指令：npm install -g hexo-cli补：cli 表示的是command line interface 也就是命令行界面
  2. 你可以先创建一个文件夹myblog，然后cd到这个文件夹下（或者在这个文件夹下直接右键git bash打开）。
  3. 接下来初始化一下hexo：hexo init
    补：hexo init 之后文件，文件夹概览
  4. 安装必要依赖项：npm install
  5. 启动hexo：hexo s
  6. 复制网址打开：http://localhost:4000/
